WordPress Options Framework

A Lightweight and easy-to-use WordPress Options Framework. It is a free framework for building theme options. Save your time!

Framework Options

Codestar Framework is built in OOP paradigm with high number of custom fields and tons of options. It's highly modern and advanced framework.

Metaboxes Options

Codestar Framework allows you to bring custom metabox settings to all of your pages and posts. We provide advanced settings with numerious number of fields.

Shortcoder Options

Codestar Framework comes with pre-built shortcode editor to manage your content. It's easy and flexible to build unlimited layouts with endless possibilites.

Customizer Options

Codestar Framework inherits the default WordPress Customizer with integration of own custom fields. It's more powerful to customize your site on live.
